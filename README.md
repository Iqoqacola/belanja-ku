## Cara Menjalankan Aplikasi Secara Lokal

1. Clone repository

   ```bash
   git clone https://github.com/Iqoqacola/belanja-ku.git
   cd belanja-ku
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Jalankan aplikasi dalam mode pengembangan

   ```bash
   npm run dev
   ```

4. Buka aplikasi di browser:  
   **http://localhost:5173/**
